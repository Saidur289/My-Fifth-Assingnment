What is the Document Object Model?

DOM is crucial for creating interactive web applications, as it bridges the gap between HTML content and JavaScript functionality.

How do you select on element from the DOM?

In DOM we select element by his id, className and the tagname.There are another useful method which is query selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of attaching listeners to multiple child elements.

How do you manipulate and elements attribute and style using the DOM?

Attributes: Use getAttribute(), setAttribute(), and removeAttribute() to manage element attributes. Styles: Use the style property or classList to change or manage styles.
